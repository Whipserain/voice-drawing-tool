# 🎨 纯语音控制绘图工具

一款完全通过语音控制的绘图工具，用户不能使用鼠标或键盘，仅通过语音指令完成全部绘图创作。

## ✨ 核心特性

### 🎤 纯语音控制
- **零鼠标/键盘依赖**：所有操作通过语音完成
- **中文语音识别**：基于 Web Speech API
- **语音反馈（TTS）**：每个操作都有语音确认
- **自动重启**：语音识别断开后自动恢复

### 🖌️ 丰富的绘图指令
- **10种图形**：圆形、矩形、三角形、五角星、爱心、箭头、椭圆、横线、竖线、直线
- **15+种颜色**：红、蓝、绿、黄、紫、粉、橙、灰、棕、青等（含深浅变体）
- **9宫格定位**：左上、上、右上、左、中间、右、左下、下、右下
- **自由画笔**：通过方向指令控制画笔移动
- **场景化绘图**：一句话画出复杂场景（雪人、星空、房子等8个场景）
- **连续方向**：「往上往右往下」一次性执行多步，「画一个Z字形」自动生成路径

### 🧠 智能命令解析
- **自然语言理解**：支持多种表达方式
- **组合命令拆解**："画三个红色圆形" → 自动拆分为3次绘制
- **复合命令**："画一个圆和一条线" → 依次执行
- **同义词容错**："蛋"→圆、"盒子"→方
- **中文数字**："三个"→3、"五个"→5

## 🚀 快速开始

### 1. 打开应用
在 Chrome 或 Edge 浏览器中打开 `frontend/index.html`。

> ⚠️ **注意**：语音识别功能仅支持 Chrome、Edge 等 Chromium 内核浏览器，需要麦克风权限。

### 2. 开始使用
1. 点击 "🎤 开始使用" 按钮
2. 允许麦克风权限
3. 说出你的绘图指令！

## 📖 语音指令指南

### 🖌️ 绘制图形
```
画一个红色圆形在中间
画一个大的蓝色矩形在左上角
画一条线从左上到右下
画一个三角形在右边
画三个黄色星星在上面
画一个绿色爱心在中间
```

### 🏔️ 场景化绘图
```
画一个雪人        → 自动生成雪人（白色圆形+眼睛+鼻子）
画一片星空        → 自动生成星空（星星+月亮+深色背景）
画一个笑脸 / 房子 / 花园 / 彩虹 / 猫 / 太阳
```

### ✏️ 自由画笔
```
开始画              → 开启自由画笔模式
往上 / 往左         → 控制画笔移动方向
往上往右往下        → 连续方向，一次性执行多步
画一个Z字形         → 自动生成Z形/方形/三角形路径
往右5步             → 方向+重复次数
停止画              → 关闭自由画笔
```

### 🎨 颜色和大小
```
红色 / 蓝色 / 绿色      → 切换颜色
深一点的红色            → 颜色混合，自动计算深红
更蓝的绿色 / 浅一点紫色  → 连续色域调节
比刚才那个再蓝一点      → 基于当前颜色相对调节
大一点 / 小一点          → 调整大小
大小30                  → 精确设置
```

### 🗑️ 删除图形
```
删除那个红色圆圈        → 按颜色+形状匹配删除
删除中间的红色圆圈      → 按位置+颜色+形状精确删除
删除最后画的            → 删除最近一个图形
```

### 📖 画布查询
```
画布上有什么            → 语音描述当前所有形状
```

### ✨ 排列美化
```
变整齐                  → 自动网格排列
居中                    → 所有图形整体居中
左对齐 / 右对齐         → 对齐
水平分布 / 垂直分布     → 等距排列
```

### 🔍 视口控制
```
放大画布 / 缩小画布     → 缩放视图
看左上角 / 看中间       → 视口平移
复原                    → 恢复默认视口
```

### 📝 文字标注
```
写上你好在中间
标注生日快乐在上面
```

### ⚡ 操作命令
```
撤销 / 后退         → 撤销上一步
重做 / 前进         → 重做
清除 / 清空         → 清除画布
保存 / 导出         → 保存为PNG图片
帮助 / 你能做什么   → 显示帮助
```

## 🛠️ 技术栈

- **HTML5 Canvas** — 画布绘图
- **Web Speech API** — 语音识别（SpeechRecognition）
- **Web Speech Synthesis API** — 语音反馈（TTS）
- **原生 JavaScript** — 无框架依赖，零外部依赖
